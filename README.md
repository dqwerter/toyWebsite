# Toy Website

Basic html, javascript, nodejs,  &amp; server setup. 

## Setup

Install nodejs:

```
curl -sL https://rpm.nodesource.com/setup_12.x | bash - #centOS 
```

Git clone then go to project root:

```
npm install
```

If missing any module, use ```npm install``` to fix.
